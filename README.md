# 프로젝트 개요

이 프로젝트는 다양한 HTML 파일들을 관리하며, 각 파일은 네이버의 주요 기능(지도, 책, 웹툰 등)을 구현하는 데 사용됩니다.

## 주요 파일 목록

- **Main.html**: 메인 페이지
- **adBanner.html**: 광고 배너 페이지
- **book.html**: 책 관련 페이지
- **calendar.html**: 캘린더 기능 추가
- **kin.html**: 지식인 기능 추가
- **mail.html**: 메일 페이지
- **mybox.html**: MYBOX
- **naverMap.html**: 네이버 지도
- **naverStore.html**: 네이버 스토어
- **news.html**: 뉴스 페이지
- **pay.html**: 페이 기능 추가
- **tv.html**: TV 관련 페이지
- **webtoon.html**: 웹툰 기능 추가
